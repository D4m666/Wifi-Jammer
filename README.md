# Wifi-Jammer
Simple wifi jammer with aircrack-ng

To install and execute the script do the following commands

  - 'git clone https://github.com/D4m666/Wifi-Jammer'
  - 'cd Wifi-Jammer'
  - 'chmod +x setup.sh'
  - 'python Wifi-Jammer.py'
 
This script only works in linux
